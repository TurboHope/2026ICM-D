### 3. Network + Performance Integration
```python
edges = pd.read_csv('data/1_lineup_network_edges.csv')
advanced = pd.read_csv('data/1_COMPLETE_advanced_stats.csv')

import networkx as nx
G = nx.Graph()

for _, row in edges.iterrows():
    G.add_edge(row['Player_A'], row['Player_B'],
               weight=row['Minutes_Together'],
               rating=row['Net_Rating'])

# Add WS as node attribute
ws_dict = advanced.set_index('PLAYER_NAME')['WS'].to_dict()
nx.set_node_attributes(G, ws_dict, 'WS')

# Find high-value, high-centrality players
centrality = nx.betweenness_centrality(G, weight='weight')
```
